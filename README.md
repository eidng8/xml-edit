A Vue.js component showing and editing XML in tree view.

## Installation

```bash
npm i -S g8-xml-edit
```

## Usage

```html
<g8-xml-edit xml="<root/>" />
<!-- OR -->
<g8-xml-edit :xml="xml" />
```
